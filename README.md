# Immunization Coverage Analysis

This project analyzes WHO immunization coverage data for 2022 using R and tidyverse.

## Tools used
- R
- tidyverse
- ggplot2

## Files included
- `analysis.R` → R script for data analysis
- `doses_by_country_antigen.png` → Bar chart of total doses by country and antigen
- `immunization-coverage_2022_MCV1.csv` → Dataset (optional)

## Description
- Frequency table of countries
- Mean and median of doses administered
- Bar chart showing doses by country and antigen
